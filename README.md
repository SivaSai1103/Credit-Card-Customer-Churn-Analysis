# Credit Card Customer Churn Analysis
## Project Overview

This project analyzes customer churn patterns in a banking environment using Power BI and DAX. The objective was to identify factors contributing to customer attrition and provide actionable recommendations to improve customer retention.
## Dataset

- Source: Kaggle
- Records: 10,127 credit card customers
- Features: Demographics, transaction behavior, credit usage, customer engagement

## Tools Used

- Power BI
- DAX
- Excel
- Data Analysis

## Key Insights

- Overall churn rate: 16.07%
- Customers with higher inactivity had greater churn tendencies.
- Lower transaction activity was associated with churn.
- Customers with fewer banking relationships were more likely to leave.
- Lower credit utilization was observed among churned customers.

## Business Recommendations

- Launch re-engagement campaigns for inactive customers.
- Offer cashback incentives to increase card usage.
- Cross-sell additional banking products.
- Improve loyalty programs for high-risk customer segments.
